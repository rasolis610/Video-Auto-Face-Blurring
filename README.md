# Video-Auto-Face-Blurring
Video Auto Face Blurring-- Face Privacy Mask
Video Face Blurring (Privacy Mask)

This is a simple Python script that automatically applies face blurring to video files for privacy protection. It can be used directly as a script or run as a standalone executable.

Description:
The tool detects faces in each frame of a video and applies a blur effect to anonymize them. It's useful for redacting personal identity in surveillance footage, interviews, or any sensitive content.
